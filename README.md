**Readme**
**1.简单说明**
一个简单的爬虫，用于爬取某站的视频

**2.环境配置**
* 在使用该爬虫之前，你需要先去下载ffmpeg，这是一个非常著名的开源项目，下载之后记得添加环境变量
* 请下载Python，这是一个开源的编程语言，然后添加环境变量
* os是标准库，无需下载，但是requests不是标准库，需要额外安装，在命令行工具中输入"pip install requests"即可安装

**3.使用的大致方法**
* 在某站找到你喜欢的视频
* 打开开发人员选项，在网络一栏可以找到视频和音频的url以及其他的，在代码里面修改即可，然后运行
